# План: Замена фона hero секции

## Задача
Заменить фоновое изображение в первой секции hero на файл `priscilla-du-preez-g86airJZ4Gs-unsplash.jpg`.

## Текущее состояние
- Hero секция использует изображение с Unsplash (строка 161):
  `url("https://images.unsplash.com/photo-1509042239860-f550ce710b93?auto=format&fit=crop&w=1800&q=80")`

## План выполнения
1. Заменить URL фонового изображения в CSS класса `.hero` на локальный файл
2. Изменить строку 161: `url("https://images.unsplash.com/photo-1509042239860-f550ce710b93?auto=format&fit=crop&w=1800&q=80")` 
   на `url("priscilla-du-preez-g86airJZ4Gs-unsplash.jpg")`

## Результат
Фон hero секции будет использовать локальное изображение `priscilla-du-preez-g86airJZ4Gs-unsplash.jpg`.
